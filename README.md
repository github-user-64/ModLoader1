# 模组加载器
这是单纯的一个模组加载器。程序和模组之间由一个公用库(PublicLibrary1)作为交互，不过程序和模组之间并没有实际的交互内容，模组在被加载后该干什么并没有要求。在主程序中只有加载模组的功能和一个模组管理器。
